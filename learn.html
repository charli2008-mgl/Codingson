HTML гэж юу вэ? (HyperText Markup Language)

HTML нь вэб хуудасны агуулга болон бүтцийг тодорхойлдог стандарт тэмдэглэгээт хэл юм.

  • HyperText: Хуудас хооронд холбоос (link) ашиглан шилжих боломж.
  • Markup: Текст, зураг, видеог вэб хөтөч дээр хэрхэн харагдахыг "tag" ашиглан тэмдэглэх.

HTML-ийн үндсэн бүтэц (Tags)

HTML нь ихэвчлэн нээх <tag> ба хаах </tag> гэсэн хос шошгоос бүрдэнэ.

<h1>Энэ бол гарчиг</h1>
<p>Энэ бол эх бичвэр буюу параграф.</p>
<a href="https://google.com">Энэ бол холбоос</a>

HTML Элемент гэж юу вэ?

HTML элемент нь Нээх tag + Агуулга + Хаах tag-аас бүрдэнэ.

1. Элементийн бүтэц

Жишээ нь: <p>Сайн байна уу?</p>

  • <p> - Нээх шошго (Start tag)
  • Сайн байна уу? - Агуулга (Content)
  • </p> - Хаах шошго (End tag)

2. Хоосон элементүүд (Empty Elements)

Зарим элементүүдэд агуулга байдаггүй бөгөөд тэдгээрийг хаах шаардлагагүй байдаг.

  • <br> - Шинэ мөрөнд шилжих
  • <img> - Зураг оруулах
  • <input> - Мэдээлэл оруулах талбар

3. Nested Elements (Үүрлэсэн элементүүд)

Элемент дотор өөр элемент байж болно.

<div>
  <h1>Гарчиг</h1>
  <p>Эх бичвэр</p>
</div>

HTML Semantics (Утга зүйт тагууд)

Semantic tag нь тухайн кодын блок ямар үүрэгтэйг хүн болон компьютерт (browser, SEO) ойлгомжтой илэрхийлдэг.

1. Non-Semantic (Утга илэрхийлдэггүй)

Ямар ч мэдээлэл өгдөггүй: <div>, <span>

2. Semantic (Утга илэрхийлдэг)

Өөрийн гэсэн утгатай: <header>, <footer>, <article>

Түгээмэл ашиглагддаг Semantic тагууд:

  • <header>: Хуудасны толгой хэсэг (лого, цэс).
  • <nav>: Цэс буюу навигацийн холбоосууд.
  • <main>: Хуудасны үндсэн, гол агуулга.
  • <section>: Сэдэвчилсэн бүлэг хэсэг.
  • <article>: Бие даасан нийтлэл, пост.
  • <aside>: Хажуугийн нэмэлт мэдээлэл (sidebar).
  • <footer>: Хуудасны хөл хэсэг (холбоо барих, зохиогчийн эрх).

CSS гэж юу вэ? (Cascading Style Sheets)

CSS нь HTML-ээр бүтээсэн элементийн харагдах байдал, загварыг тодорхойлдог хэл юм.

  • Layout: Элементүүдийн байршил, хоорондын зай.
  • Design: Өнгө, фонт, зураг, хүрээ (border).
  • Responsiveness: Гар утас, таблет, компьютер дээр вэбийг өөр өөрөөр харагдуулах.

CSS-ийн үндсэн бүтэц (Syntax)

CSS нь Selector болон Declaration block-оос бүрдэнэ.

h1 {
  color: blue;        /* Текстийн өнгө */
  font-size: 24px;    /* Үсгийн хэмжээ */
  text-align: center; /* Голлуулах */
}

CSS Selectors (Сонгогч)

CSS Selector нь загваржуулахыг хүсэж буй HTML элементээ "барьж авах" буюу сонгох арга юм.

1. Element Selector

HTML тагийн нэрээр нь шууд сонгох. Бүх ижил тагуудыг өөрчилнө.

p { color: blue; } /* Бүх параграф цэнхэр болно */

2. Class Selector

Тусгайлан нэрлэсэн хэсгийг сонгохдоо . (цэг) ашиглана. Олон элементэд ижил нэр өгч болно.

.highlight { background: yellow; }

3. ID Selector

Цор ганц элементийг сонгохдоо # (тор) ашиглана. Нэг хуудас дээр зөвхөн нэг л давтагдашгүй ID байна.

#main-title { font-size: 40px; }

4. Universal Selector

Хуудас дээрх бүх элементийг сонгохдоо * ашиглана.

* { margin: 0; padding: 0; }

CSS Layout (Байршил тогтоох)

Layout нь вэб хуудасны элементүүдийг хаана, ямар зайтай байрлахыг шийддэг.

1. Flexbox (Нэг хэмжээст)

Элементүүдийг нэг мөрөнд (хөндлөн) эсвэл нэг баганад (босоо) эгнүүлэн байрлуулахад тохиромжтой.

.container {
  display: flex;
  justify-content: space-between; /* Зайг ижил хуваах */
  align-items: center;            /* Босоо тэнхлэгээр голлуулах */
}

2. CSS Grid (Хоёр хэмжээст)

Мөр болон баганыг зэрэг ашиглан илүү нарийн, цогц бүтэц үүсгэхэд ашиглана.

.wrapper {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 3 ижил багана */
  gap: 20px;                          /* Хоорондын зай */
}

JavaScript (JS) гэж юу вэ?

JavaScript нь вэб хуудсыг интерактив буюу "амьд" болгодог програмчлалын хэл юм.

  • Интерактив байдал: Товчлуур дарах, форм бөглөх, цэс нээгдэх зэрэг бүх хөдөлгөөн.
  • Өгөгдөл боловсруулах: Хэрэглэгчийн оруулсан мэдээллийг тооцоолох, сервер рүү илгээх.
  • Бодит цагийн шинэчлэл: Хуудсыг дахин ачаалахгүйгээр мэдээллийг шинэчлэх (жишээ нь: сошиал медиагийн "like" дарах).
"HTML бол бүтэц, CSS бол загвар, харин JavaScript бол үйлдэл юм."

JavaScript Functions (Функц)

Функц нь тодорхой нэг үйлдлийг гүйцэтгэхэд зориулагдсан кодын блок юм.

1. Функц зарлах (Declaration)

function түлхүүр үг ашиглан нэр өгч үүсгэнэ.

function sayHello() {
  console.log("Сайн байна уу!");
}

2. Параметр ба Аргумент (Parameters & Arguments)

Функц руу мэдээлэл дамжуулж, илүү уян хатан болгох боломжтой.

function greet(name) { // 'name' бол параметр
  console.log("Сайн байна уу, " + name);
}
greet("Бат"); // "Бат" бол аргумент

3. Утга буцаах (Return Statement)

Функц тооцоолол хийгээд хариуг нь return ашиглан буцааж өгдөг.

function add(a, b) {
  return a + b;
}
let result = add(5, 3); // result нь 8 болно

JavaScript Data Types

JavaScript variables can hold different data types. They are divided into Primitives and Objects.

  • String: Текст өгөгдөл (жишээ нь, "Hello").
  • Number: Бүхэл болон аравтын тоо (жишээ нь, 42, 3.14).
  • Boolean: Үнэн эсвэл худал логик (жишээ нь, true, false).
  • Undefined: Утга оноогдоогүй хувьсагч.
  • Null: Санаатайгаар хоосон утга.
  • Object: Нарийн бүтэцтэй өгөгдлийн цуглуулга.

JavaScript Variables (Хувьсагч)

Хувьсагч нь өгөгдлийг (текст, тоо гэх мэт) түр хадгалах сав юм. JavaScript-д хувьсагч зарлах 3 үндсэн арга бий.

  • let: Утга нь дараа өөрчлөгдөж болох хувьсагч үүсгэхэд ашиглана.
  • const: Утга нь хэзээ ч өөрчлөгдөхгүй (тогтмол) хувьсагч үүсгэхэд ашиглана.
  • var: Хуучин арга (орчин үеийн кодонд бага ашиглагддаг).

Жишээ код:

let age = 25;
age = 26; // let ашигласан тул утгыг өөрчилж болно

const name = "Бат";
// name = "Болд"; // Алдаа заана! Учир нь const өөрчлөгдөхгүй.

JS Conditions (Нөхцөл шалгах)

Тодорхой нөхцөл биелсэн эсэхээс хамаарч програм өөр өөр үйлдэл хийх шаардлага гардаг. Үүнд if...else бүтцийг ашиглана.

if...else бүтэц:

let score = 85;

if (score >= 90) {
    console.log("Онцгой сайн!");
} else if (score >= 80) {
    console.log("Сайн байна.");
} else {
    console.log("Дахин хичээгээрэй.");
}

JS DOM (Document Object Model)

DOM-ийг ашиглан JavaScript нь таны HTML элементүүдийг олж харж, тэдгээрийг өөрчлөх, устгах эсвэл шинээр нэмэх боломжтой болдог.

  • Элемент сонгох: document.querySelector(".class-name") ашиглаж HTML-ээс элемент барьж авна.
  • Үйлдэл сонсох: addEventListener() ашиглан хэрэглэгч товч дарах, хулгана чирэх үед юу болохыг заана.

Жишээ: Товчлуур дарахад текст өөрчлөгдөх

// 1. HTML доторх товч болон текстийг барьж авах
const btn = document.querySelector(".my-btn");
const text = document.querySelector(".my-text");

// 2. Товчлуур дээр дарах (click) үйлдэл нэмэх
btn.addEventListener("click", function() {
    text.textContent = "Текст амжилттай өөрчлөгдлөө!";
    text.style.color = "#38bdf8";
});
